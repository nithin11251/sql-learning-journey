# 1. What is Data?

Data is raw facts and figures.
Example:
7369, SMITH, 800

When data is processed, it becomes information.

---

# What is SQL?

SQL (Structured Query Language) is used to:
- Store data
- Retrieve data
- Update data
- Delete data

SQL works with relational databases like Oracle.

---

# What is DBMS?

DBMS (Database Management System) is software used to store and manage data.

Examples:
- Oracle
- MySQL
- PostgreSQL

DBMS provides:
- Data security
- Backup
- Multi-user access
- Data consistency
